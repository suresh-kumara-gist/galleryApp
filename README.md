# Gallery App for Redmi Note 7

A high-performance, offline gallery app optimized for Redmi Note 7 and similar devices.

## Features
- 📸 View all photos
- 🎥 Play videos
- 📄 Preview documents (PDF, DOC, TXT, PPT, XLS)
- 🔍 Search files
- ⭐ Favorites system
- 🗂️ Folder view
- 🎨 Material Design with dark mode
- ⚡ Optimized for Snapdragon devices

## Build Instructions

### Local Build
```bash
# Clone repository
git clone <repository-url>

# Build debug APK
./gradlew assembleDebug

# Build release APK
./gradlew assembleRelease

# Run tests
./gradlew test

# Run lint checks
./gradlew lint
